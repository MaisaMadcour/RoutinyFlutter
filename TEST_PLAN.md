# 🧪 خطة اختبار تطبيق روتيني (Routiny)

> قائمة تاسكات لاختبار التطبيق كله قبل النشر. علّمي ✅ على كل تاسك بعد ما تتأكدي منه.
> ركّزي بشكل خاص على القسم (٢) و(٣) لأنهم أحدث التعديلات.

---

## 0) تجهيز قبل الاختبار
- [ ] ركّبي آخر نسخة release على جهاز حقيقي.
- [ ] جرّبي على **موبايلين على الأقل**: واحد شاشة صغيرة/قديمة، وواحد حديث.
- [ ] غيّري **حجم الخط في إعدادات النظام** لأكبر قيمة → افتحي التطبيق وتأكدي إن الخط **ما اتغيرش** والكروت ما اتداخلتش.
- [ ] جرّبي والجهاز فيه نت وبدون نت (الإعلانات لازم ما توقّفش التطبيق).

---

## 1) عام / تنقّل (Navigation & General)
- [ ] الأيقونة في الدروَر = أيقونة روتيني البرتقالية الصح.
- [ ] كل التابات بتفتح: روتيني، عناية، تايم، اختبار، إعدادات.
- [ ] زر الباك من أي تاب → يرجّع لتاب **روتيني** الأول، وتاني باك → يخرج من التطبيق.
- [ ] اتجاه الشاشة RTL سليم في كل الصفحات (مفيش نص مقلوب).
- [ ] البانر الإعلاني ظاهر **تحت** شريط التنقل في **كل** التابات (بما فيها تايم).

---

## 2) التايمر والبومودورو (الأهم) ⏱️
### تشغيل أساسي
- [ ] تاب تايم: التبديل بين "بومودورو" و"تايم" شغّال.
- [ ] تغيير المدة من الإعدادات بيتحفظ.
- [ ] "ابدأ التركيز/التايم" بيفتح شاشة العدّاد.
- [ ] العدّاد بينزل ثانية بثانية صح.

### شاشة القفل والإشعار (ركّزي هنا)
- [ ] أول ما يبدأ التايمر → يظهر إشعار في **الستاتس بار** فيه الأرقام كبيرة بخط Montserrat بألوان وردي/تركواز/أبيض.
- [ ] الأرقام وزر "إيقاف" **في النص أفقياً** في الإشعار.
- [ ] زر "إيقاف" في الإشعار بيوقف التايمر فعلاً.
- [ ] **اقفلي الشاشة** والعدّاد شغّال → الإشعار ظاهر على **شاشة القفل** والأرقام بتنزل صح.
- [ ] افتحي القفل قبل ما يخلص → الوقت **مظبوط على الزمن الحقيقي** (مش متجمّد/متأخّر).
- [ ] اقفلي الشاشة وسيبي الوقت يخلص وانتِ قافلة → يجيلك إشعار **"خلص التايم ⏱️"** (أو "خلصت جلسة البومودورو 🍅") بصوت واهتزاز.
- [ ] الضغط على إشعار الانتهاء يفتح التطبيق.

### البومودورو متعدد الدورات
- [ ] دورة بومودورو كاملة: تركيز → استراحة → تركيز… بتتنقل صح والإشعار بيتحدّث.
- [ ] في آخر دورة بس بيطلع إشعار الانتهاء (مش في نص الدورات).

### الموسيقى والإضافات
- [ ] اختيار صوت تركيز بيشتغل، وبيفضل شغّال بعد قفل الشاشة.
- [ ] الضغط المطوّل لإنهاء الجلسة بدري → يخرج **من غير** إشعار "خلص الوقت".
- [ ] زر التدوير/القلب (flip) بيكبّر الأرقام.

### الإحصائيات
- [ ] صفحة إحصائيات التركيز بتفتح.
- [ ] المحور الرأسي ثابت بالساعات: يوم/أسبوع=12، شهر=42، سنة=336.
- [ ] المحور الأفقي: يوم=ساعات (12ص/3ص…)، أسبوع=أيام، شهر=**4 أسابيع**، سنة=شهور.
- [ ] التنقّل بين الفترات (السهمين) بيغيّر البيانات صح.

---

## 3) صفحة روتيني والمهام ✅
- [ ] المهام الافتراضية ظهرت أول تشغيل.
- [ ] علّمي مهمة فرعية → بتتعلّم، والخط **تحت الجملة** (مش فوقها).
- [ ] إكمال كل المهام الفرعية → كونفيتي + تتسجّل كمنجزة.
- [ ] **اختبار التاريخ:** علّمي مهام النهاردة، بعدين روحي ليوم تاني (ماضي/مستقبل) → لازم يبقى **فاضي/غير متعلّم**.
- [ ] ارجعي لليوم الأصلي → علاماتك لسه موجودة.
- [ ] إنشاء مهمة جديدة من الـ FAB (+) بيشتغل ويظهر فوراً.
- [ ] ضغطة مطوّلة على كارت → تظهر أزرار تعديل/مسح.
- [ ] تعديل مهمة بيتحفظ.
- [ ] مسح مهمة → أنيميشن المسح ثم تختفي.
- [ ] الكالندر الأسبوعي + اختيار التاريخ من الـ date picker شغّالين.
- [ ] زر "اليوم" يرجّع لليوم الحالي.

---

## 4) صفحة العناية 🌿
- [ ] الاقتباس اليومي بيفتح.
- [ ] الكاسة (الميّة) ظاهرة **فوق شريط التنقل مباشرة**.
- [ ] ضغط الكاسة يفتح صفحة شرب الميّة.
- [ ] بانر التنفّس + كل أقسام المقالات بتظهر.
- [ ] فتح أي مقال عناية → يقرأ صح، وزر المشاركة شغّال.
- [ ] قفل المقال → ممكن يظهر إعلان (كل 3 دقايق).

---

## 5) شرب الميّة 💧
- [ ] إضافة (نص كاسة / كاسة / كاسة كبيرة / كمية مخصوصة) بتزوّد الميّة والكاسات.
- [ ] **زر التراجع** بينقص آخر كمية اتضافت (والكاسات بتنقص).
- [ ] أنيميشن الكاسة (مستوى الميّة) بيتحرّك صح.
- [ ] كل يوم جديد الكاسة تبدأ **فاضية** (ما بتخزّنش ميّة اليوم اللي فات).
- [ ] جدول آخر 7 أيام صح + الستريك (🔥) لو فيه أيام متتالية.
- [ ] إعدادات الميّة (الهدف/حجم الكاسة/التذكير) بتتحفظ.
- [ ] زر الإعدادات وسهم الباك ثابتين في الهيدر.

---

## 6) الانعكاس / حاسة بإيه 💭
- [ ] خطوة المزاج: لازم تختاري مزاج عشان "التالي" يشتغل.
- [ ] خطوة المشاعر: لازم **3 مشاعر + 3 مؤثرات** بالظبط.
- [ ] خطوة الكتابة + تفعيل "احتفظي بمذكراتي".
- [ ] الانتقال للنتيجة → ممكن يظهر **إعلان (إنترستيشيال)** قبلها (كل 5 دقايق).
- [ ] النتيجة: الإمباثي، وصف اليوم، اللمحة، النصيحة، الروتين، المقال المقترح.
- [ ] "اعتمدي الروتين" بيشتغل، والمقال المقترح بيفتح مقال عناية مناسب.
- [ ] لو فعّلتي المذكرات → النص بيتسجّل في مذكراتي.

---

## 7) الاختبارات 📝
- [ ] شبكة الاختبارات بتظهر بألوان صح.
- [ ] فتح اختبار → شاشة المقدمة.
- [ ] شاشة الأسئلة: الكارت والإجابات **ما بيتداخلوش** (جرّبي على موبايل صغير) — وتعمل scroll لو لزم.
- [ ] اختيار إجابة → ينتقل للسؤال اللي بعده تلقائي.
- [ ] زر الباك بيرجّع سؤال للخلف.
- [ ] آخر سؤال → شاشة المعالجة → ممكن **إعلان** → النتيجة.
- [ ] نتيجة الاختبار صح حسب الدرجة.

---

## 8) البروفايل 👤
- [ ] فتح البروفايل من صورة المستخدم في صفحة روتيني.
- [ ] تعديل الاسم بيتحفظ.
- [ ] محرّر الأفاتار (تكبير/تحريك الصورة) + الحفظ.
- [ ] إحصائيات الإحساس + المزاج الغالب صح.
- [ ] كالندر التنفّس بيعلّم الأيام اللي عملتي فيها تمرين.
- [ ] **مذكراتي مقفولة** (🔒) → ضغطة → رسالة الإعلان → بعد الإعلان تتفتح **للأبد**.
- [ ] بعد الفتح: ضغطة تانية تفتح المذكرات مباشرة (من غير إعلان).
- [ ] المذكرات المحفوظة بتظهر، والمسح شغّال.

---

## 9) الإعدادات ⚙️
- [ ] كل صفوف الإعدادات بتظهر صح (تصميم مسطّح).
- [ ] أي toggle/إعداد بيتحفظ بعد إعادة فتح التطبيق.

---

## 10) الإشعارات 🔔
- [ ] صفحة الإشعارات بتفتح.
- [ ] (لو ممكن) تذكير شرب الميّة بيوصل في وقته.
- [ ] الإذن POST_NOTIFICATIONS بيتطلب أول تشغيل.

---

## 11) الإعلانات 📺 (تأكيد عام)
- [ ] البانر شغّال (Test Ad) في كل التابات.
- [ ] الإنترستيشيال بيظهر في: نهاية التايمر، نتيجة الاختبار، نتيجة الانعكاس، قفل المقال، زر الدعم.
- [ ] الإعلانات ما بتظهرش بشكل متلاحق مزعج (فيه فاصل زمني).
- [ ] إعلان المكافأة (مذكراتي) بيشتغل.
- [ ] لو مفيش نت/إعلان مش جاهز → التطبيق بيكمّل عادي من غير ما يعلّق.

---

## 12) ثبات وأداء (Stability)
- [ ] افتح/اقفل التطبيق كذا مرة → مفيش كراش.
- [ ] دوران الشاشة (لو مسموح) ما بيكسرش التخطيط.
- [ ] استهلاك البطارية معقول أثناء التايمر على القفل.
- [ ] إعادة تشغيل الجهاز → التطبيق بيفتح عادي (BootReceiver).
